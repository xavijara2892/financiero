# Xavier Finance Pro Lite

Versión modular y más ligera de la calculadora financiera.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Estructura
- `core/`: lógica financiera
- `ui/`: interfaz Streamlit
- `utils/`: formato y exportación
- `data/`: valores por defecto

## Deploy
Compatible con Streamlit Community Cloud y Render.
